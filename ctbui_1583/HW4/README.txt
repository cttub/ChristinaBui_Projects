To Run This program.
Go into Terminal and go to directory that VirtualPet.java file is in. 
Compile program by typing javac VirtualPet.java
Then run by typing java VirtualPet.

BUGS: Right now there is error input for playerInteraction. So in order to properly play game, enter in shown options instead of typing in another number. If you type in another option other than 1 2 or 3. THERE WILL BE A PUNISHMENT.
